Definition

Free assembly of customized blocks is part of the fun of deep learning, if not all the essence of it. We study the two-stage computation as follows: \[ x \mathop{\longrightarrow}_{h} z \mathop{\longrightarrow}_{g} y \] where \(h\) and \(g\) are two different computation blocks which are applied to the input \(x\) sequentially. so $ = f(x) = g(h(x)) $. \(z = h(x)\) here can be understood as an intermediate label

The key question of end-to-end symbolic grounding problem can thus be stated as follows:

Given a ground-truth dataset \((X,Z,Y)\) representing the input, the intermediate label and the final label, suppose in training the intermediate label \(Z\) is absent, and we are only using \((X,Y)\) to minimize $_{x,y} L(g(h(x)) - y) $, can we obtain the intermediate label \(z\)?
